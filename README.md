# Portafolio de QA Automation

Este repositorio reúne una serie de proyectos diseñados para demostrar mis habilidades en **Quality Assurance (QA)**, tanto en pruebas manuales como en la transición hacia **automatización de pruebas**.  

## Objetivo
Mostrar mi capacidad para:
- Diseñar y ejecutar casos de prueba automatizados.
- Validar aplicaciones web y APIs con herramientas modernas.
- Integrar pruebas en pipelines de CI/CD.
- Documentar proyectos de manera clara y profesional.

## Tecnologías utilizadas
- **Lenguajes:** Python
- **Frameworks de pruebas:** Selenium, Pytest
- **Pruebas de API:** Requests, Postman
- **Control de versiones:** Git y GitHub
- **CI/CD:** GitHub Actions (integración futura)

## Proyectos incluidos
1. **Proyecto 1: Automatización de Login con Selenium**  
   Flujo de autenticación en un sitio demo, validando credenciales y mensajes de éxito.

2. **Proyecto 2: Flujo de compra en e-commerce (Selenium)**  
   Automatización de búsqueda, carrito y checkout en una tienda demo.

3. **Proyecto 3: Pruebas de API**  
   Validación de endpoints públicos con Pytest y Requests.

4. **Proyecto 4: Integración CI/CD**  
   Ejecución automática de pruebas en GitHub Actions.

## Cómo usar este portafolio
Cada proyecto está organizado en su propia carpeta con:
- Código fuente
- Archivo `requirements.txt` para instalar dependencias
- Documentación en `README.md` con instrucciones de ejecución
